## Premise
This project was originally just Pokémon Red/Green on the Terminal, plain and simple, but it kind of slowly derived into a original Pokémon game (branching from Red and Green) with its own premise, followed by a mini engine that I wanted people to use (if anyone wants to make Pokémon Games on their terminal, of course...). However, development is mainly focused on the game until either finishing the entire story and side content, or if a bug/meta pops up that makes the game unplayable/not fun.


## How to download and play
- Website: https://nigelisguy.github.io/pokethon/web/index.html
- Go to releases, click download for your system
- Run pokethon/main.py in a terminal

## Modding
- Go to devtools, and open any of the tools needed, should be pretty self explanatory, but do note that SOME OF THESE TOOLS ARE UNFINISHED, AND MIGHT HAVE COMPLETE UI OVERHAULS, as well as the fact they are pretty buggy
