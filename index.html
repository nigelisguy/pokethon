About

Version: 0.6

POKéTHON is an early-build Pokémon-style project with experimental features and multiplayer elements.

Note for Early Build Testers

If you have any build that doesn't include save functionality (pre-0.6), delete it before updating. For future builds, transfer your save file between versions.

Controls

  • Z = Enter
  • X = Cancel
  • C = Menu
  • Arrow Keys = Movement
