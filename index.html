POKéTerminal

A Pokémon adventure in your terminal!

About

Version: 0.6

POKéTerminal is an early-build terminal game that is based on, you guessed it, Pokémon.

Note

Since the save files are constantly changing formats, the save file is not compatable between versions for now. This will be fixed after the demo comes out

Controls

  • Z = Enter
  • X = Cancel
  • C = Menu
  • Arrow Keys = Move
